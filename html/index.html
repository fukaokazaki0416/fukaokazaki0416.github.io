About私について

岡崎 楓可 - Okazaki Fuka

1998年4月16日東京都生まれ岡崎楓可と申します。 大学卒業後、キャリアショップの代理店で勤務していましたが、事業縮小のため退社。新たなキャリアを考える中で、過去に自分が助けられた経験から医療事務の仕事に興味を持ち、2年間医療機関で働きました。その経験を通じて、人を支える仕事のやりがいを感じました。 しかし、私の中には常に「人に届けるワクワクを形にしたい」という思いがあり、WEBデザインという新たな挑戦に向けて、職業訓練校でWEBデザインやコーディングを学び、基本スキルを習得しました。 これからは、使いやすさと美しさを兼ね備えたデザインを通じて、多くの人に喜びやワクワクを届けられる仕事をしていきたいと考えています。

Skill出来ること

（01）Design

Webサイト、バナー、Instagramデザインの制作ができます。 商品やサイトイメージに合った配色にこだわり作品を作っています。 また、テイストが近いデザインを調べ、情報収集を行うことでよりよいデザインを作れるようにしています。

Tools : Photoshop/Illustrator/XD/Canva

（02）Coding

HTML・CSSを使用し、デザイン通りのコーディングをすることができます。 また、Javascriptを使用し、サイトに動きを出すことで表現できる幅を広げることができます。 コードはほかの人が見てもわかるように意識し、コメントを加えたり、整列させたり等工夫をしています。

Tools : VisualStudioCode/DreamWeaver
